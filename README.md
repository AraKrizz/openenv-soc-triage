# CyberGuard SOC Triage Environment
This is an OpenEnv reinforcement learning environment designed to evaluate an AI agent's ability to act as a SOC Analyst.

## Features
* Realistic JSON-based network states.
* Dynamic scoring based on speed of resolution.
* Multiple difficulty tiers (Easy, Medium, Hard).
